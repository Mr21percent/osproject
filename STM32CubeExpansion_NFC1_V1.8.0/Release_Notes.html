Release Note for X-CUBE-NFC1 Software for STM32, Expansion for STM32Cube

Copyright 2014 STMicroelectronics

 

This software is providing NFC capabilities thanks to M24SR device, running on STM32. It is built on top of STM32Cube software technology that ease portability across different STM32 microcontrollers.

X-CUBE-NFC1 software features:
    • Complete middleware to build applications using our dynamic NFC tag (M24SR).
    • Easy portability across different MCU families thanks to STM32Cube
    • Interaction with all NFC capable smartphone
    • Free user-friendly license terms
    • Examples implementation available on board X-NUCLEO-NFC01A1 plugged on top of one NUCLEO board.
                       
         

Related information and documentation:
    • AN4624: Getting started with the STM32 Nucleo and the M24SR expansion board X-NUCLEO-NFC01A1, available at www.st.com
    • STM32Cube: www.st.com/stm32cube
    • STM32 Nucleo boards: www.st.com/stm32nucleo
    • STM32 Nucleo expansion boards: www.st.com/x-nucleo
